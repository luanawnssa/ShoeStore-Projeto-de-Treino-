Este repositório contém um projeto simples desenvolvido para praticar HTML e CSS, incluindo criação de páginas, organização de layout, uso de grid responsivo, cartões de produtos, formulários e conceito básico de acessibilidade.
O site simula uma pequena loja de sapatos com três páginas principais:

Início

Produtos

Contato

O foco do projeto é treinar estrutura semântica, estilização moderna, responsividade e boas práticas de desenvolvimento frontend.
